# @lawminded/web

Next.js App Router frontend for LawMinded.

## Environment Variables

- `NEXT_PUBLIC_APP_URL`
- `NEXT_PUBLIC_API_URL`
- `NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY`
- `NEXT_PUBLIC_COOKIE_POLICY_VERSION`

## Commands

```bash
pnpm --filter @lawminded/web dev
pnpm --filter @lawminded/web build
pnpm --filter @lawminded/web start
```
